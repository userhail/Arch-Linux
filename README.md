# Arch Linux Installation

## 📅 03. Juli 2026

Heute habe ich **Arch Linux** erfolgreich **komplett manuell** installiert – **ohne den `archinstall`

Dabei habe ich unter anderem:

* die Festplatte manuell partitioniert,
* die Partitionen formatiert und eingehängt,
* das Basissystem mit `pacstrap` installiert,
* die `fstab` erstellt,
* Zeitzone, Sprache, Hostname und Benutzer eingerichtet,
* den Bootloader installiert und konfiguriert,
* das Netzwerk eingerichtet
* und das System erfolgreich gestartet.

Die Installation hat mir einen tieferen Einblick in den Aufbau und die Funktionsweise von Arch Linux gegeben. Anstatt einen automatischen Installer zu verwenden, habe ich jeden Schritt selbst durchgeführt und mich dabei an der offiziellen Arch Wiki orientiert.

> **Kein `archinstall` – nur das Terminal und die Arch Wiki. Und ein bisschen Chatgpt**
